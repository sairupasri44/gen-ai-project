# Project Prerequisites
## FinRelief AI — AI Powered Debt Relief and Financial Recovery Platform

---

## 1. Project Overview

FinRelief AI is a full stack AI-powered web application designed to assist individuals
facing financial distress. The platform accepts a user's monthly income, total outstanding
debt, and monthly expenses, then computes a financial health score, predicts the probability
of successful debt settlement, and delivers personalized recovery advice generated by
Google Gemini AI — all through a clean, dark-themed browser interface.

| Property       | Detail                                          |
|----------------|-------------------------------------------------|
| Project Name   | FinRelief AI                                    |
| Project Type   | Full Stack Web Application with AI Integration  |
| Purpose        | Debt Relief and Financial Recovery Assistance   |
| Frontend       | React.js 19 with Vite 8                         |

| Database       | SQLite 3 (built into Python)                    |
| AI Integration | Google Gemini 2.5 Flash API                     |

Install all of them with a single command after activating the virtual environment.

### 3.5 External Services and API Keys

| Service           | Purpose                                        | Where to Get                           |
|-------------------|------------------------------------------------|----------------------------------------|
| Google Gemini API | Generates AI financial advice and strategies   | https://aistudio.google.com/app/apikey |

**How to configure your API key:**

1. Visit https://aistudio.google.com/app/apikey and generate a free API key
2. Create a file named `.env` inside the `backend/` folder
3. Add the following line to it:

```
GEMINI_API_KEY=your_google_gemini_api_key_here
```

> Never commit the `.env` file to Git. It is already listed in `.gitignore` to
> prevent accidental exposure. Share a `.env.example` file instead which contains
> only a placeholder value and is safe to commit.

## 4. Environment Setup Instructions

### Step 1 — Open the Project in VS Code

```bash
cd gen-ai-project
code .
```

### Step 2 — Set Up the Python Backend

```bash
cd backend

# Create a virtual environment
python -m venv venv

# Activate on Windows
venv\Scripts\activate

# Activate on macOS / Linux
source venv/bin/activate

# Install all backend dependencies
pip install -r requirements.txt
```

### Step 3 — Configure the Gemini API Key

```bash
# Create the .env file inside backend/
# Add your key:
GEMINI_API_KEY=your_google_gemini_api_key_here
```

### Step 4 — Start the FastAPI Backend

```bash
# Inside backend/ with venv active
uvicorn main:app --reload
```

Backend runs at: `http://localhost:8000`
Swagger API docs at: `http://localhost:8000/docs`

### Step 5 — Set Up and Start the React Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend runs at: `http://localhost:5173`

### Step 6 — Open the Application

```
http://localhost:5173
```

Register a new account, log in, enter your financial data, and click Analyze.
